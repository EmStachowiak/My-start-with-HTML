DOCTYPE

Dichiara che il documento è di tipo HTML5


HTML

Il tag HTML è l'elemento principale di una pagina html(tutto è contenuto all'interno)


questo è link dispense e file del corso

Title

Elemento che specifica un titolo per la pagina HTML ( che viene mostrato nella barra del titolo nel browser o nella scheda della pagina)


Body

Elemento che definisce il corpo del documento, è un contenitore per tutti i contenuti visibili, come intestazioni, paragrafi, immagini, collegamenti ipertestuali(link), tabelle,elenchi, etc..

h1 e p

Con h1, h2, h3, h4, h5, h6, definiamo le intestazioni (come il titolo do un articolo o il nome di un prodotto o servizi

con p invece definiamo un paragrafo, come ad esempio la descrizione di un articolo o prodotto

Immagini HTML

Le immagini HTML sono definite con il tag.

Il file sorgente ( src), il testo alternativo ( alt), widthe heightsono forniti come attributi:
img src="nomeimmagine.estensione" height="50%" width="50%"
